# Changelog
All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/)
and this project adheres to [Semantic Versioning]
(https://semver.org/spec/v2.0.0.html).

## Unreleased

## [3.1.0] - 2024-01-10
### Changed
- Updated survey data to November 2023
- Updated README file

## [3.0.0] - 2023-01-26
### Changed
- Removed "taxon_code" columns from survey data and replaced with 
"scientific_name" and "description" columns. 
- Updated survey data to November 2022
- Updated Data Dictionary with above changes

### Removed
- Removed example figures for joining and summarizing
- Removed Rocky Intertidal taxa data sheet 

## [2.0.0] - 2022-01-11
### Changed
- Updated survey spreadsheet with data up to November 2021

## [1.0.0] - 2020-03-04
### Added
- Creative Commons license
- README file
- Overall Rocky Intertidal protocol
- Reference figure for summaries & joins
- Reference script for summaries & joins

### Changed
- Updated survey spreadsheet with 2019 survey data
- Updated Data Dictionary
- Updated taxa spreadsheet with new codes and structure

### Removed
- Rocky Intertidal site data
- Barnacle abundance protocol
- Survey data from 2017-2018

## [0.1.0] - 2019-02-15
### Added
- Barnacle Abundance survey data (2016-Nov. 2018)
- Rocky Intertidal taxa codes
- Rocky Intertidal site data
- Data Dictionary with definitions of all variables
- Barnacle Abundance protocol
